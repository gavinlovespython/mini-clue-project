# configuration system

this file defines adjustable settings for the clue engine.

## engine settings

(default values chosen for simplicity and readability)

- allow_duplicates: false
- max_length: 120
- random_seed: none
- output_format: "text"
- logging: false

## supported languages

these languages are available for future formatting, demos, or code‑style clues.

- python
- javascript
- html
- css
- rust
- fortran (experimental / for fun)

## future settings

(these are ideas for later expansion)
