# configuration system

this file defines adjustable settings for the clue engine.

## engine settings

(note: these defaults are chosen for simplicity)

- allow_duplicates: false
- max_length: 120
- random_seed: none
- output_format: "text"

(these are not implemented yet, just planned ideas)
